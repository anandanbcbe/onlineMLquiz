☁Free Database — Firebase Setup

This stores student answers in the cloud so you can see them on your laptop even when students submit from their phones. One-time setup, free forever, no credit card.

1

Create Firebase project

Go to console.firebase.google.com → Add project → Give any name → Continue → Continue → Create project

2

Enable Realtime Database

In your project → left sidebar → Build → Realtime Database → Create database → choose any location → Start in test mode → Enable

3

Copy your Database URL

After enabling, you'll see a URL like:
https://your-project-default-rtdb.firebaseio.com
Copy that entire URL.
